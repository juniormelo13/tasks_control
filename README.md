## ğŸ“Œ Description

TasksControl is a web application that provides a **fluid, intuitive, and responsive experience**, allowing the user to not only create, edit, and remove tasks, but also **manage deadlines, filter information with precision, and customize the interface** to their preference.<br> All tasks and settings are automatically saved to **Local Storage**, ensuring data persistence even after reloading or closing the page. Expired tasks are **visually highlighted**, alerting about missed deadlines.

## ğŸš€ Setup and Local Execution

To run TasksControl in your development environment, follow the steps below:

**1. Prerequisites:**

Before you begin, ensure you have the following tools installed:

* [Git](https://git-scm.com)
* [Node.js](https://nodejs.org/en/) (version 22.20.0 or higher recommended)
* Additionally, it's good to have an editor to work with the code, such as [VSCode](https://code.visualstudio.com/)

**2. Clone this Repository:**

```bash
git clone https://github.com/juniormelo13/tasks_control.git
```

**3. Access the project folder in your terminal:**

```bash
cd tasks_control
```

**4. Install the dependencies:**

```bash
npm install
```

**5. Start the development server:**

```bash
npm start
```
This will start the server (usually at http://localhost:3000) and open the application in your default browser. The server features Hot Module Replacement (HMR) for automatic updates during development.

**6. To generate the optimized production build (in the /dist folder), run:**

```bash
npm run build
```
